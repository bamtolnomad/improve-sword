### 목표

- 첫 30초 세션에서 1~10단계 빠른 강화 감각을 확인한다.
- 10~15단계에서 판매/분해/보관 판단이 생기는지 확인한다.
- 보호석/축복석은 아직 BM이 아니라 고위험 구간 개입 수단으로만 검증한다.

### 적용한 변경

- 시작 골드: `5,000G` -> `12,000G`
- 보호석 가격: 강화석 `20` -> `15`
- 축복석 가격: 강화석 `35` -> `25`
- 분해석 지급 앞당김:
  - 7->8: `0` -> `1`
  - 8->9: `0` -> `1`
  - 9->10: `1` -> `2`
  - 10->11: `2` -> `3`

### 의도

- 첫 플레이가 +7 근처에서 끊기면 강화 게임의 핵심 감각을 보기 전에 세션이 닫힌다.
- 분해가 9단계 이전에는 의미가 거의 없어 판매와 분해의 선택지가 늦게 열린다.
- 보호석/축복석 가격은 실제 BM 전 단계이므로, 테스트 중에는 조금 더 자주 만져볼 수 있어야 한다.

### 다음 확인

- 기본 시작으로 +10 도달이 너무 쉬운지 직접 플레이로 확인한다.
- 12~15단계에서 판매/분해/보관이 실제로 갈리는지 본다.
- 23단계 이후 축복석 전략이 거의 발동하지 않는 문제는 다음 패스에서 고단계 비용/확률 곡선을 별도로 다룬다.

```bash
npm run playtest:report
```

### 판단

- 기본 세션은 +10 도달 감각을 확인하는 용도로는 괜찮다.
- 기본 세션만으로는 +12~15의 판매/분해/보관 선택을 검증할 수 없다.
- 12~15 선택 검증은 `10만G 밀기` 또는 수동 QA 골드 지급 후 별도 세션으로 보는 편이 맞다.
- +10 분해 반복은 보호석까지 너무 멀다. 분해의 의미를 첫 세션에 보여주려면 보호석 가격을 더 낮추거나, 10단계 전후 분해석을 더 줘야 한다.
- +10 보관 반복은 GPS 증가가 보이지만, 즉시 강화 재시도에는 도움을 주지 않는다. 방치 보상 수령 경험과 묶어서 평가해야 한다.

### 다음 후보

- 기본 세션 목표를 `+10 도달`로 명확히 유지한다.
- 2차 세션 프리셋을 `10만G 시작`으로 두고 +12~15 의사결정을 테스트한다.
- 보호석 구매가 너무 멀면 다음 패스에서 보호석 가격 `15 -> 10` 또는 9~11단계 분해석 추가를 비교한다.

### 후속 적용

- QA 패널에 `2차 세션` 버튼을 추가했다.
- 이 버튼은 상태를 새 세션으로 초기화하되 플레이 노트는 유지하고, `100,000G` 시작점에서 +12~15 판매/분해/보관 선택을 바로 테스트하게 한다.

### 판단

- 2차 세션에서 +12는 거의 확정 도달 구간이고, +15는 약 50% 도달 구간이다.
- +12 판매/분해/보관은 “상위 도전 포기 후 자원화”라서 선택지는 명확하지만, +15 도전과 양립하지 않는다.
- +12 보관은 GPS 증가가 가장 선명하다. 방치 보상 수령 경험을 함께 보여주면 보관의 의미가 잘 산다.
- 보호석은 현재 2차 세션에서는 너무 늦다. +12 분해를 반복해도 평균 보호석 구매가 0.51개에 그치고 +15 도전까지 이어지지 않는다.
- 다음 밸런스 패스는 보호석 가격 `15 -> 10` 또는 +12 전후 분해석 상향을 비교하는 편이 맞다.
