# 검키우기 제작용 게임 기획서

## 0. 기획 상태

- 문서 버전: v1.0
- 작성 기준: Codex와 Gemini 게임 기획자 협업 루프 3차 개정
- 최종 완성도 평가: 9.2 / 10
- 현재 결론: 제작용 GDD로 확정 가능. 단, 확률/경제 수치는 프로토타입 QA로 조정한다.

## 1. 게임 한 줄 정의

검키우기는 방치로 모은 골드와 재료를 사용해 검을 강화하고, 강화된 검을 판매/분해/보관하며 더 높은 강화 단계에 도전하는 캐주얼 방치형 강화 게임이다.

## 2. 핵심 방향

- 플레이어가 가장 신경 쓰는 행동은 전투 조작이 아니라 강화다.
- 방치 보상은 별도 메인 콘텐츠가 아니라 강화 시도를 계속하게 만드는 연료다.
- 예전 한국 플래시게임식 검 강화 감성, 즉 단순한 버튼과 성공/실패/파괴의 긴장감을 현대 모바일 방치 구조로 확장한다.
- 검은 경험치 레벨이 아니라 강화 성공 횟수로 성장한다.
- 현재 1~30단계 검 에셋은 `sword_images/sword_stage_01.png` ~ `sword_stage_30.png`로 준비되어 있다.

## 3. 타깃 유저

- 짧은 시간에 성장과 확률 결과를 확인하고 싶은 캐주얼 유저
- 강화 도박, 리스크 관리, 익절 판단을 즐기는 유저
- 복잡한 조작 없이 서브 게임처럼 켜두는 방치형 유저
- 옛 플래시게임 감성의 단순하고 반복적인 성장 게임을 선호하는 유저

## 4. 핵심 재미

1. 강화 버튼을 누르기 직전의 긴장감
2. 팔지 말고 한 번 더 강화할지 결정하는 익절/탐욕의 갈등
3. 파괴 후에도 분해석, 마일리지, 방치 보상으로 다시 도전하는 회복 루프
4. 검 이미지가 단계별로 성장하며 고급스러워지는 시각적 보상
5. 판매/분해/보관 선택을 통해 다음 도전을 준비하는 자산 운용감

## 5. 전체 루프

1. 방치 보상 또는 검 판매로 골드 획득
2. 골드로 검 강화 시도
3. 결과 확인: 성공, 유지, 하락, 파괴
4. 성공한 검을 계속 강화할지 판매/분해/보관할지 결정
5. 판매로 골드 확보, 분해로 강화석 확보, 보관으로 GPS 증가
6. 강화석과 광고 보상으로 보호석/축복석 등 보조 아이템 확보
7. 고위험 구간에 재도전
8. 30단계 달성 시 최소 환생으로 다음 회차 진입

## 6. 세션 루프

### 30초 세션

- 접속
- 방치 보상 수령
- 1~10단계 안전/저위험 구간 빠른 강화
- 실패하면 바로 재시도

### 3분 세션

- 10~18단계 도전
- 판매/분해/보관 판단
- 보호석을 살지, 골드로 더 밀지 결정

### 10분 세션

- 도감/보관 상태 확인
- 광고 보상 수령
- 보호석/축복석을 동원해 고단계 강화
- 파괴 시 원혼 마일리지 확인 후 다음 목표 재설정

### 방치 복귀 세션

- 최대 12시간까지 오프라인 보상 누적
- 광고 시청 시 방치 보상 2배
- 보상은 바로 강화 시도 횟수로 환산되는 구조로 표시

## 7. 강화 결과

- 성공: 검 단계 +1
- 유지: 단계 변화 없음, 비용만 소모
- 하락: 검 단계 -1
- 파괴: 현재 검 소멸, 분해석 일부와 파괴 마일리지 획득
- 보호: 보호석 사용 시 파괴를 1회 방지, 하락은 막지 않음

## 9. 판매/분해/보관

### 판매

- 목적: 골드 확보
- 추천 상황: 다음 강화 시도 횟수를 늘리고 싶을 때
- 감정: 지금 팔면 이득, 더 누르면 대박
- 설계 의도: 익절 판단을 만든다.

### 분해

- 목적: 강화석 확보
- 추천 상황: 보호석/축복석 구매가 필요한 중후반
- 감정: 성장을 멈추고 안전장치를 마련하는 선택
- 설계 의도: 고단계 강화 준비 루프를 만든다.

### 보관

- 목적: 도감/GPS 증가
- 추천 상황: 최초 도달 검 또는 방치 수익을 늘리고 싶을 때
- 감정: 당장 돈은 못 벌지만 장기적으로 계정이 강해짐
- 설계 의도: 방치형 리텐션을 만든다.

## 10. 재화

| 재화 | 주요 획득처 | 주요 소모처 |
|---|---|---|
| 골드 | 방치 보상, 검 판매, 광고 2배 보상 | 강화 비용 |
| 강화석 | 검 분해, 파괴 보상, 이벤트 | 보호석, 축복석 구매 |
| 다이아 | 과금, 업적, 랭킹 보상 | 복구권, 패키지, 편의 기능 |
| 원혼 마일리지 | 15단계 이상 파괴 | 천장 발동 |
| 환생 카운트 | 30단계 달성 후 환생 | 영구 보너스 |

## 11. 방치 보상 공식

```text
GPS = (BaseGold + CollectionBonus + StoredSwordBonus) * (1 + RebirthMultiplier)
OfflineGold = GPS * OfflineSeconds
```

- BaseGold: 기본 초당 골드
- CollectionBonus: 도감 등록 단계 수 기반 보너스
- StoredSwordBonus: 보관 중인 검 단계에 따른 보너스
- RebirthMultiplier: 환생 1회당 +50%
- 오프라인 보상 최대 누적: 기본 12시간, 월정액 24시간
- 광고 시청 시 해당 복귀 보상 2배

## 12. 하루 플레이 모델

### 1일차 목표

- 목표 도달: 10~12단계
- 플레이 감정: 강화가 잘 붙고 빠르게 성장한다.
- 경제 비중: 방치 보상 80%, 판매 20%
- 주요 학습: 강화, 판매, 도감

### 3일차 목표

- 목표 도달: 16~18단계
- 플레이 감정: 첫 하락/파괴를 경험한다.
- 경제 비중: 방치 보상 50%, 판매 50%
- 주요 학습: 분해, 보호석, 원혼 마일리지

### 7일차 목표

- 목표 도달: 22~25단계
- 플레이 감정: 계획적 도전과 보호석 관리가 중요해진다.
- 경제 비중: 방치 보상, 보관 GPS, 광고 보상이 모두 중요
- 주요 학습: 축복석, 풀도핑, 고단계 익절

## 14. 실패 마일리지/천장 정책

- 정책: 완전 공개
- 명칭: 부러진 검의 원혼
- 노출 위치: 15단계 이상 메인 강화 버튼 하단
- 표기: `부러진 검의 원혼 35 / 100`
- 누적 조건: 파괴 시에만 누적
- 누적량: 강화 테이블의 파괴 마일리지 값
- 100 도달 시 원혼 폭주 발동
- 원혼 폭주 효과: 다음 강화 1회 파괴 확률 0%, 성공 확률 2배
- 이유: 확률형 시스템 신뢰와 법적/운영 리스크를 고려해 숨기지 않는다.

- 현재 강화 진행 상태

- 도감 달성도
- 다이아
- 보조 아이템
- 환생 카운트

- 환생 1회당 전체 강화 성공 확률 +0.5%p

### 설계 의도

복잡한 스킬 트리 없이 최소 구현으로 30단계 이후 리텐션을 방어한다. 이후 업데이트에서 초월석, 환생 노드, 31~40단계 확장으로 발전시킨다.

## 21. 주요 리스크와 해결

| 리스크 | 설명 | 해결 |
|---|---|---|
| 연속 파괴 스트레스 | 유저 이탈 가능 | 원혼 마일리지 완전 공개, 파괴 보상 지급 |
| 보호석 남발 | 강화 긴장감 붕괴 | 광고/상점 일일 제한 |
| 과금 돌파 | 콘텐츠 수명 단축 | 복구권 주간 제한, 같은 검 1회 제한 |
| 30단계 이후 허무함 | 코어 유저 이탈 | 최소 환생 MVP 포함 |
| 자동화 어뷰징 | 탭 수익 악용 | 탭 골드 일일 캡, 방치/판매 중심 경제 |
| 표기 신뢰 문제 | 확률형 게임 반감 | 성공/유지/하락/파괴 확률 UI 표시 |

## 22. 남은 논의 사항

1. 게임의 비주얼 톤: 완전 플래시게임 감성인지, 현대 모바일 캐주얼 톤인지
2. 판매가 공식: 누적 투자금 회수형인지, 단계 고정가형인지
3. 도감 보너스: 최초 등록만 보상할지, 보관 수량도 반영할지
4. 복구권 과금 강도: 주간 3회가 적절한지
5. 출시 시점의 환생 보상: 단순 확률/GPS 보너스만 둘지, 최소 노드 3개를 둘지

## 23. 현재 결정 요약

- 강화 중심 정체성 유지
- 방치는 강화 연료 공급
- 판매/분해/보관은 각각 골드/강화석/GPS로 분리
- 원혼 마일리지는 완전 공개
- 환생은 MVP에 최소 형태로 포함
- 최종 문서 점수는 9.2점으로 제작 착수 가능
